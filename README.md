学习小程序大致分三个阶段  
1）概念建立阶段，通过浅阅官文文档，阅读社区教程、文章完成  
2）知识结构充实阶段，通过阅读他人的demo源码，在本地运行，调试修改，带着问题学习是这个阶段的主要特征，要善用搜索引擎，善于在群内提问  
3）开发修行阶段，投入到实际的项目开发中去，在实践中提高认识，并提出新的问题，改进既有的开发方案，编写小程序类库、工具等
